My end of year project, a simple satellite tracking web application offering the possibility to browse hundreds of satellites as well as tracking their positions real-time. <br>
Config: <br>
##auth <br>
-create new firebase authentication project <br>
-generate key <br>
-create service.json in the satellite server file and add your credentials <br>
##run flask server : <br>
python app.py (make sure to install all required packages) <br>
##run React App : <br>
npm start (make sure to install all dependencies (npm install)) <br>
Have fun :)) <br>
